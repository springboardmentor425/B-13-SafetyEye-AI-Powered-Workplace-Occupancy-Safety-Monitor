# Support

## Getting Help

Use the following channels:

- Usage/setup problems: open a GitHub issue.
- Bug reports: use the bug report issue template.
- Feature ideas: use the feature request issue template.
- Security concerns: follow `SECURITY.md`.

## What to Include

When asking for help, include:
- Operating system and shell.
- Python version.
- Exact command used.
- Full error output.
- Steps already attempted.

## Documentation First

Before opening an issue, check:
- `README.md`
- `resources-and-scripts/USAGE.md`
- `resources-and-scripts/OIDv4_ToolKit/README.md`
