# Security Policy

## Supported Versions

This project is in active development. Security fixes are prioritized on the default branch.

| Version | Supported |
| --- | --- |
| main | Yes |
| older snapshots | No |

## Reporting a Vulnerability

If you discover a security issue:

1. Do not disclose it publicly in an issue.
2. Use GitHub private vulnerability reporting for this repository when available.
3. If private reporting is unavailable, open an issue with minimal details and request a private contact channel.

## Response Expectations

- Initial triage target: within 5 business days.
- A fix timeline will depend on severity and impact.
- Coordinated disclosure is expected after mitigation is available.
